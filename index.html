Weather Search

Brief:

Using a 3rd party API to collect data, in this case data about the weather for a location you "Search". Save the Searched location for future searches and display the weather details for the current day and also display a five day weather forecast, highlighting the severity of the UV Index

Saved locations are stored in local storage but ony the last ten searches are presented back for another Search. Built a button to toggle the display of the forecasted weather and ensured that any city that was entered for Searcing was not saved to the top ten Searches again. The solution makes use of the open-weather-api to source its data.
